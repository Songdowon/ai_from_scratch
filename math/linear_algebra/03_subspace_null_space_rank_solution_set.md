# 📌 [Subspace - Null Space - Rank - Solution Set of $Ax=b$]

## 1. Flow

- In the previous topic, we studied how vectors generate a space through **span**, **basis**, and **column space**.
- Now the focus shifts from generating a space to understanding its **structure** and the **conditions for the existence of solutions**.
- This leads naturally to **subspace**, **null space**, **rank**, and the **solution set of $Ax=b$**.

> span / basis / column space → subspace / null space / rank / solution set of $Ax=b$
---

## 2. Interpretation

Summary: **"after generating spaces, we study their structure and the behavior of solutions"**

- A **subspace** is a smaller vector space inside a larger one.
- The **null space** is the set of all vectors mapped to zero by a matrix.
- The **rank** tells us the dimension of the column space of a matrix.
- The **solution set of $Ax=b$** depends on whether the target vector $b$ can be generated by the columns of $A$.

---

## 3. Mathematical Expression

### (1) Subspace

$$
0 \in W
$$

$$
u, v \in W \Rightarrow u+v \in W, \qquad
$$

$$
c \in \mathbb{R},\; u \in W \Rightarrow cu \in W
$$

A subspace is closed under addition and scalar multiplication.

---

### (2) Null Space

$$
\mathrm{Nul}(A) = \{x \mid Ax = 0\}
$$

The null space is the set of all vectors mapped to zero by $A$.

---

### (3) Rank

$$
\mathrm{rank}(A) = \dim(\mathrm{Col}(A))
$$

The rank is the dimension of the column space.

---

### (4) Solution Set of $Ax = b$

$$
Ax = b
$$

A solution exists if and only if $b \in \mathrm{Col}(A)$.

---

## 4. How It Connects to the Flow

- Let $A$ be an $m \times n$ matrix. Its column vectors lie in $\mathbb{R}^m$.
- These columns generate the **column space** of $A$, which is a subspace of $\mathbb{R}^m$.
- The number of linearly independent columns is the **rank** of $A$, so rank tells us how large the reachable output space is.

This is why rank is directly connected to the solvability of

$$
Ax=b
$$

- A solution exists only when $b$ lies in the column space of $A$.
- In other words, $b$ must be reachable as a linear combination of the columns of $A$.
- If $\operatorname{rank}(A)=m$, then the columns of $A$ span all of $\mathbb{R}^m$, so $Ax=b$ has a solution for every $b \in \mathbb{R}^m$.
- If $b \notin \operatorname{Col}(A)$, then the system has no solution.

When a solution does exist, the **null space** explains the structure of the solution set.

- The null space consists of all vectors $x$ such that $Ax=0$.
- If $\operatorname{Nul}(A)=\{0\}$, then the solution is unique.
- If a solution exists and the null space contains nonzero vectors, then there are infinitely many solutions, because vectors in $\operatorname{Nul}(A)$ can be added to a solution without changing the output.

So the two spaces play different roles:

- **column space** determines whether a solution exists
- **null space** determines how many solutions there can be

These are connected by the rank-nullity relation:

$$
\operatorname{rank}(A) + \operatorname{nullity}(A) = n
$$

This means that the input dimension is divided into two parts:

- directions that contribute to the output
- directions that are mapped to zero

## 5. Connection to AI 🔗

- In AI, a matrix transformation maps input vectors to output vectors.
- From this perspective, these concepts help explain what information is preserved, what is lost, and whether a target output is reachable.

This idea appears in several important AI applications:

- **representation learning** → learning meaningful vector spaces from data
- **dimensionality reduction** → keeping important directions while removing redundancy
- **information compression** → preserving essential structure in a smaller space
- **model expressiveness** → understanding how much a transformation can actually represent
- **solvability** → understanding whether a target output can be produced

In this sense, these concepts connect linear algebra to questions of **representation, compression, expressiveness, and solvability** in AI models.

---

## 6. Connection with Other Concepts

These ideas naturally lead to the next viewpoint in linear algebra.

After studying the structure of spaces and the behavior of solutions, the next step is to study matrices as **transformations between spaces**.

- **linear transformation / linear mapping** → viewing $Ax$ as a function from one vector space to another
- **kernel** → the transformation viewpoint of the null space
- **image / range** → the transformation viewpoint of the column space
- **one-to-one / onto** → properties explained by null space and column space

So, subspace, null space, rank, and solution sets describe how the behavior of a matrix appears in linear systems, while the next topic describes that same behavior from the viewpoint of mappings between spaces.
