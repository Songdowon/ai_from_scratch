# 📌 [Span - Basis - Column Space]

## 1. Flow

- In the previous topic, we moved from a **linear system** to a **vector equation**, and then to a **matrix equation**.
- In a vector equation, a target vector is written as a **linear combination** of given vectors.
- Once we see vectors in terms of linear combinations, the next question is: **what space can these vectors generate?**
- This leads naturally to **span**, **linear independence**, **basis**, **dimension**, and **column space**.

---

## 2. Interpretation

Summary: **"vectors do not only appear in equations — they also generate spaces"**

- **span** is the set of all vectors that can be generated by linear combinations of given vectors.
- **linear independence** means that none of the vectors is redundant.
- **basis** is a set of vectors that spans a space and is linearly independent.
- **dimension** is the number of vectors in a basis.
- **column space** is the span of the columns of a matrix.

---

## 3. Mathematical Expression

### (1) Linear Combination

$$
x_1 a_1 + x_2 a_2 + \cdots + x_n a_n
$$

One particular combination of vectors using scalar coefficients.

---

### (2) Span

**Span**$\lbrace a_1, a_2, \dots, a_n \rbrace$  
$= \lbrace x_1 a_1 + x_2 a_2 + \cdots + x_n a_n \mid x_1, x_2, \dots, x_n \in \mathbb{R} \rbrace$

The set of all vectors that can be generated from the given vectors.

---

### (3) Linear Independence

$$
x_1 a_1 + x_2 a_2 + \cdots + x_n a_n = 0
\Rightarrow
x_1 = x_2 = \cdots = x_n = 0
$$

The vectors are independent if only the trivial combination produces the zero vector.

---

### (4) Column Space

 $A = [a_1\ a_2\ \cdots\ a_n]$  

**Col**$(A)$ = **span**$\lbrace a_1, a_2, \dots, a_n \rbrace$

The column space is the set of all linear combinations of the columns of $A$.

---

## 4. How It Connects to the Flow

A **linear combination** describes one particular vector formed from given vectors.  
If we consider not just one combination, but all possible combinations, we get the **span**.  
However, a spanning set may still contain unnecessary vectors. This is where **linear independence** becomes important, because it tells us whether each vector contributes a genuinely new direction.  
When a set of vectors both spans a space and is linearly independent, it becomes a **basis**. In this sense, a basis is the most efficient set of vectors for describing a space.  
The number of vectors in a basis gives the **dimension**, which tells us how many independent directions the space has.  
Finally, when these ideas are applied to a matrix, the span of its columns becomes the **column space**, which describes all vectors that the matrix can generate.

So the flow becomes:

> linear combination → span → linear independence → basis → dimension → column space

---

## 5. Connection to AI 🔗

- In AI, data is often represented as vectors in a **feature space**.
- A model transforms input vectors into new representations, so linear algebra is not only about individual vectors, but also about the **spaces generated by data and transformations**.
- From this perspective, concepts such as **span**, **linear independence**, **basis**, and **column space** help us understand how information is represented, transformed, and compressed.

This idea appears in many important AI concepts:

- **representation learning** → learning useful vector spaces that capture important patterns in data
- **embedding spaces** → mapping data such as words, images, or users into vectors so that similar items are located close to each other
- **latent spaces** → learning hidden spaces that contain compact and meaningful features of the original data

In this sense, many AI models can be viewed as systems that map data from one vector space to another while learning which directions are important and which information is redundant.

---

## 6. Connection with Other Concepts

These ideas naturally lead to other important concepts in linear algebra.

Once we understand how vectors generate a space, the next step is to study the structure of that space more closely.

- **subspace** → a space that is contained within another vector space
- **rank** → the dimension of the column space
- **null space** → the set of vectors mapped to zero by a matrix
- **solution set of $Ax=b$** → depends on whether $b$ lies in the column space of $A$

So, span, basis, and column space help explain **what a system can generate**, while later concepts help explain **the internal structure of those spaces and the behavior of solutions**.
